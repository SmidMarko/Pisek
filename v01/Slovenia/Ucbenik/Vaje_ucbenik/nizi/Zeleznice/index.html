Železnice

Na Slovenskih železnicah so naredili končni pregled ostanke tirnic. Na začetku leta so imeli n metrov tirnic, ki so jih čez leto polagali glede na potrebe. Njihove dolžine v metrih (cela števila) so si shranjevali v niz, kjer so med seboj ločene s podpičji. Napiši funkcijo Razlika(n,s), ki kot parameter dobi tak niz, ter vrne dolžino proge, ki je ostala. Primer: Razlika(56,"10;2;25") naj vrne 19.
