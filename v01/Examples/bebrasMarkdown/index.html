Jeu des variables

Rappel : syntaxe des affectations

L'affectation `x=e` est une instruction qui consiste à : * évaluer la valeur de l'expression `e` (dans un premier temps ce sera une expression arithmétique pouvant, bien sûr, inclure d'autres variables). * ranger cette valeur dans une zone de la mémoire * mémoriser que la variable `x` désigne cette zone de la mémoire Ainsi, l'affectation `x=2+1` va évaluer le membre droit (ici `2+1`), ranger le résultat (ici 3) en mémoire, et mémoriser que la variable `x` désigne la zone mémoire où ce `3` a été rangé. Par abus de langage, on dit que `x` *reçoit* la valeur 3. Pour l'instant, une variable peut être vue comme le nom d'une boîte dans laquelle on stocke un nombre. Par exemple, l'instruction `a=a+1` calcule d'abord la valeur à droite du `=` en ajoutant 1 au nombre stocké dans la variable `a` puis range le résultat dans cette même variable `a`. Ce qui revient à augmenter la valeur de `a` de 1. Commencez par étudier les exemples ci-dessous pour vérifier si vous avez compris. Dans la partie "Difficile", vous verrez qu'en Python on peut modifier plusieurs variables à la fois (c'est une affectation multiple) : on calcule d'abord toutes les valeurs à droite du `=` puis on les range (dans l'ordre) dans les variables à gauche du `=`.
